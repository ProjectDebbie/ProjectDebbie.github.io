Database

The Database of Experimental Biomaterials and their Biological Effect (DEBBIE) is a new open access tool which will enable a more efficient and evidence-based selection of materials for medical applications. The database aims to curate exhaustive inventories of biomaterials and their associated data using text mining tools. It can be used to recreate or update reviews of the literature in a more rapid and repeatable manner as well as generate a comprehensive map of research activity and findings.

The DEBBIE Database contains annotations from over 200 PubMed Abstracts related to the field of Biomaterials. Each annotation is labelled by one or more of the following categories:

  • Biomaterial
  • Structure
  • Application
  • Manufactured Object
  • Manufactured Object Features
  • Bioactive Substance
  • Tissue
  • Medical Application
  • Cell
  • Effect on Biological System
  • Associated Biological Process
  • Research Technique

Ontology

The Devices, Experimental scaffolds and Biomaterials Ontology (DEB) was developed to facilitate information curation in the area of medical devices, experimental scaffolds and biomaterials. The ontology was designed to complement existing medical vocabularies, with emphasis on material processing, structures and features associated with biomaterials and experimental scaffolds. It was developed using text mining tool and a systematic approach.
