Ontology

The Devices, Experimental scaffolds and Biomaterials Ontology (DEB) is an open resource for organizing information about biomaterials, their design, manufacture and biological testing. It was developed using text analysis for identifying ontology terms from a biomaterials gold standard corpus, systematically curated to represent the domain’s lexicon. Topics covered by DEB were validated by members of the biomaterials research community. The ontology may be used for searching terms, performing annotations for machine learning applications, standardized meta-data indexing and other cross-disciplinary data exploitation. If you are a biomaterials scientist, we encourage and welcome your input to this effort! You can flag terms and errors through the 'issues' tab in DEB's GitHub repository.

Database

The Database of Experimental Biomaterials and their Biological Effect (DEBBIE) is a new open access tool which will enable a more efficient and evidence-based selection of materials for medical applications. The database aims to curate exhaustive inventories of biomaterials and their associated data using text mining tools. It can be used to recreate or update reviews of the literature in a more rapid and repeatable manner as well as generate a comprehensive map of research activity and findings.

The DEBBIE Database contains annotations of PubMed abstracts in field of biomaterials, including implants, medical devices, and exeprimental scaffolds. Each annotation is labelled by one or more of the following categories:

  • Biomaterial
  • Medical Application
  • Study Type
  • Manufactured Object
  • Associated Biological Process
  • Tissue Type
  • Structure
  • Effect on Biological System
  • Cell Type
  • Material Processing
  • Adverse Effects
  • Animal Model
  • Manufactured Object Features
  • Biologically Active Substance
  • Research Technique
